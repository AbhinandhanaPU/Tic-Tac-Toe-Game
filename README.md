# Flutter Tic Tac Toe

Flutter Tic Tac Toe is a simple mobile application developed using the Flutter framework. It allows two players to play the classic Tic-Tac-Toe game on their mobile devices.

## Features

- **Two Player Mode:** Play against a friend on the same device.
- **Simple Interface:** Clean and intuitive UI design for easy gameplay.
- **How to Play:** Provides instructions on how to play the game.
- **Win Detection:** Automatically detects winning combinations and declares the winner.
- **Draw Detection:** Detects when the game ends in a draw.
- **Reset Option:** Option to reset the game board and start a new game.
- **Restart Option:** Option to restart the game and start a new game.

## How to Play

1. **Home Screen:**
   - The home screen displays two buttons: "Start Game" and "How to Play".
   - Clicking on "Start Game" button takes you to the player name input screen.
   - Clicking on "How to Play" button provides instructions on how to play the game.

2. **How to Play Screen:**
   - This screen explains the rules and mechanics of the game.
   - It provides instructions on how players can take turns marking empty squares to win the game.

3. **Player Name Input Screen:**
   - This screen prompts players to enter their names before starting the game and it is optional to enter their names.
   - Once the names are entered, clicking on "Start Game" button takes you to the game screen.

4. **Game Screen:**
   - The game screen displays the Tic Tac Toe grid where players can make their moves.
   - Players take turns marking empty squares with their respective symbols (X or O).
   - The game continues until one player achieves a winning combination or the board is filled (resulting in a draw).
